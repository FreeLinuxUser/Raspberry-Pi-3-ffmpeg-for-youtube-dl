#How to use

Download ffmpeg.sh to your Raspberry Pi,use sudo tu run,

git clone https://github.com/FreeLinuxUser/Raspberry-Pi-3-ffmpeg-for-youtube-dl/

chmod +x ffmpeg.sh

sudo ./ffmpeg.sh

It can takes 2 hours,so run this via VNC is a good idea.
